Invoice 1 is the first new assignment type for creating a sales receipt. It asks the students to create a table listing 5 items, quantities, and prices. Students must also write HTML and Javascript to find the total by calculating subtotal and tax. All of the computing must be done dynamically and organized by comments.

On my first attempt, I struggled with how to limit the decimal calculations and place variables within a string. It helped to quickly do a run-through of the code to refresh my memory on the correct code. Its also a good idea to have your first attempt saved in a folder just in case I struggle to remember how to do the exercise.
